# BeachBook - Technologies

| Technology | Purpose |
| --- | --- |
| Expo (React Native) | Mobile app framework and runtime |
| Expo Router | File-based navigation and routing |
| TypeScript | Static type safety |
| Jazz (`jazz-tools`) | Local-first relational database with sync and permissions |
| Uniwind | Tailwind-style layout and styling for React Native |
| `@lodev09/react-native-truesheet` | Bottom sheet UI |
| `@tabler/icons-react-native` | Icon system |
| `@shopify/flash-list` | High-performance list rendering |
| `react-native-svg` | SVG rendering support |
| `react-native-calendars` | Calendar/date UI components |
| React Hook Form | Form state handling |
| Zod | Schema and form validation |
| SurveyJS (via `use-dom`) | Protocol form engine in WebView/DOM context |
| date-fns | Date parsing, formatting, and utilities |
| `expo-secure-store` | Secure key-value storage for sensitive data (tokens, credentials) |
| `expo-crypto` | Cryptographic primitives (hashing, random bytes) |
| `expo-camera` | Camera access for photo/video capture |
| `@react-native-community/datetimepicker` | Native date and time picker UI |
| `tailwind-variants` | Variant-based component styling with Tailwind class composition |
| `tailwind-merge` | Merges conflicting Tailwind class names safely |
| `react-native-pager-view` | Native swipeable page/tab view primitive |
| `react-native-tab-view` | Tab navigation UI built on pager-view |
| `react-native-signature-canvas` | Signature capture canvas via WebView |
| `@clerk/expo` | Authentication and user management library for Expo |
| `@clerk/localizations` | Localization keys/translations for Clerk UI |
| `@hookform/resolvers` | Validation resolvers (e.g. Zod) for React Hook Form |
| `react-native-get-random-values` | Polyfill for `crypto.getRandomValues` in React Native |
| `react-native-qrcode-svg` | QR code generator component for React Native |
